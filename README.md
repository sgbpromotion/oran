# SGB Promotion — website (Vite + React + Tailwind)\n\nQuick start (locally)\n1. `npm install`\n2. `npm run dev` (open http://localhost:5173)\n\nBuild\n- `npm run build`, then `npm run preview` to test the production build.\n\nDeploy to Vercel\n1. Push this repo to GitHub/GitLab/Bitbucket.\n2. In Vercel, create a New Project and import the repo.\n3. Build command: `npm run build` — Output directory: `dist`\n4. For custom domain, add `sgbpromotion.com` in Vercel > Domains and follow their DNS instructions.\n
